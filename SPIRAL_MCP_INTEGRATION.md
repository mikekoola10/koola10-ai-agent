# Spiral MCP Integration

Spiral utilizes the Model Context Protocol to manage autonomous asset acquisition and research.

## TypeScript Client

The integration uses the `@modelcontextprotocol/sdk`.

### Setup

Ensure dependencies are installed:

```bash
npm install @modelcontextprotocol/sdk
```

### Execution

Run the client using `ts-node`:

```bash
npx ts-node spiral-mcp-client.ts
```
