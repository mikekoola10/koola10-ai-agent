# Koola10 Omega: Parent-Friendly AI Setup

Koola10 Omega is a specialized instance of the Koola10/Spiral ecosystem designed specifically for parents and non-technical users.

## 1. Accessing the Omega Dashboard
The Omega dashboard can be accessed by appending `?role=omega` to the main application URL:
- **URL:** `http://<your-app-domain>/?role=omega`

## 2. Key Features
- **Simplified Interface:** Large fonts, high contrast, and clear action buttons.
- **Emergency Help:** A prominent red button that triggers immediate alerts to family members.
- **Health Monitoring:** Background swarms monitor motion and safety, alerting Alpha (you) if anomalies are detected.
- **Bill Payments:** Simplified one-tap bill payment interface linked to the shared economic ledger.

## 3. Configuration
- **User Role:** `omega`
- **Associated Swarms:** Health, Finance, Safety, Communication.

## 4. Troubleshooting
- If the dashboard shows "CHECK SYSTEM", verify that the home sensors (e.g., motion, energy) are online and reporting data.
- For non-technical users, we recommend setting the Omega URL as a "Home Screen" bookmark on their tablet or smartphone.
