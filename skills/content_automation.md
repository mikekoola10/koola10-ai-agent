# Content Automation Skill
Multi-platform content scheduling.

## Description
Generates and schedules social media content across multiple platforms.

## Required Tools
- `content` (Swarm)
- `social_media`

## Trigger Conditions
- New product release or milestone.
- Scheduled weekly content plan.

## Step-by-Step Workflow
1. **Creation**: `content` swarm generates platform-specific posts.
2. **Review**: Visual and text review for brand alignment.
3. **Scheduling**: `social_media` tool schedules posts for optimal times.
4. **Monitoring**: Track engagement metrics after posting.

## Expected Output Format
`{ "posts_scheduled": 10, "platforms": ["LinkedIn", "Twitter"] }`
